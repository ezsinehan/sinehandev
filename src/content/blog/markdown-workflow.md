I used markdown files with frontmatter so new posts do not require component edits.

The checklist is tiny:

1. add a file in `src/content/blog/`
2. fill title/date/summary/tags
3. write body

That is enough for list pages, tag views, and post detail routing. Future me can add complexity later without losing this baseline.
